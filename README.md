React buat [keluarBerapa-api](https://keluarberapa.herokuapp.com).

## Step

Langkah untuk run keluarBerapa:

### Clone Repo

````
git clone https://github.com/abdurrahmanjaisy/keluarBerapa.git
````
### npm install dependencies

````
npm install
````
### mulai npm

````
npm start
````
